Расписание

Проактивные сообщения в чат MAX по триггерам: события из базы (дни рождения и т.п.), интервал, напоминание до события. Выполнение — воркер Celery Beat раз в минуту. Для типов DATABASE и REMINDER загрузите события (CSV/JSON). Время указывается по Саратову (Europe/Saratov, UTC+4), если в ISO-строке нет явного смещения (+03:00, Z и т.д.).

Новое расписание

Интервал отсчитывается от реального времени; тик планировщика привязан к часовому поясу Саратов (UTC+4).

Момент события в файле — по Саратову (UTC+4), если в дате нет часового пояса.

Для ежегодных дат в JSON события укажите annual: true (или ezhegodno: true). Даты в CSV/JSON без смещения считаются саратовским временем.
